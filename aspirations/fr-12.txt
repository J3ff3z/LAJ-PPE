			<h1 class="entry-title">Péché et rédemption</h1>		</header><!-- .entry-header -->		<div class="entry-meta">

<h5 class="wp-block-heading">Définitions</h5>



<p>Étymologiquement, le péché est une chute hors de la voie. C&rsquo;est une transgression (à la loi religieuse), une  désobéissance (à Dieu).</p>



<p>La rédemption, du latin &laquo;&nbsp;<em>redemptio&nbsp;&raquo;, le rachat, </em>désigne dans le vocabulaire théologique chrétien l&rsquo;acte par lequel Jésus rachète les hommes esclaves de leurs péchés en le payant de sa vie. J&rsquo;essaierai d&rsquo;expliciter ce concept plus bas. Augustin d&rsquo;Hippone (354-430), connu sous le nom de saint Augustin, a écrit : &laquo;&nbsp;<em>Heureuse faute qui nous a valu un tel rédempteur</em>&laquo;&nbsp;.</p>



<h5 class="wp-block-heading">Le péché dans le judaïsme</h5>



<p>Le judaïsme n&rsquo;insiste pas sur la notion de péché. Seul le péché rituel est pris en compte, et il est traité paternellement. Donc, rater un sabbat ou ne pas manger kascher ne conduit pas en enfer. <br>Un jour de l&rsquo;année est particulier, c&rsquo;est le Yom Kippour, le jour du grand pardon. Lors d&rsquo;un jeûne de 25 heures entrecoupé de 5 prières, chaque juif demande que ses fautes envers Dieu lui soit pardonnées. Les fautes envers les hommes, elles, doivent être réparées, elles ne sont jamais pardonnées.</p>



<p>Dans des temps &laquo;&nbsp;<em>anciens</em>&laquo;&nbsp;, lors du Yom Kippour, le grand prêtre choisissait deux boucs, l&rsquo;un était sacrifié à Dieu, l&rsquo;autre emportait tous les péchés d&rsquo;Israël dans le désert. C&rsquo;est le rite du &laquo;&nbsp;<em>bouc émissaire</em>&nbsp;&raquo; relaté dans le livre du Lévitique 16, 15-22. Voici le texte des versets 20 à 22 (on est dans le désert, au temps de Moïse et de son frère Aaron) :</p>



<blockquote class="wp-block-quote is-layout-flow wp-block-quote-is-layout-flow"><p>Une fois achevée l&rsquo;expiration du sanctuaire, il fera approcher le bouc encore vivant. Aaron lui posera les deux mains sur la tête et confessera à sa charge toutes les fautes des Israélites, toutes leurs transgressions et tous leurs péchés. Après en avoir ainsi chargé la tête, il l&rsquo;enverra au désert sous la conduite d&rsquo;un homme qui se tiendra prêt et le bouc emportera sur lui toutes leurs fautes en un lieu aride.</p></blockquote>



<p>On ignore si cette cérémonie a réellement été appliquée. Les fautes dont il est question, sont des fautes collectives, les manquements du peuple envers Dieu, on ne parle pas de fautes personnelles.</p>



<h5 class="wp-block-heading">Le péché dans l&rsquo;islam</h5>



<p>Dans l&rsquo;islam, toute transgression de la loi est assimilée à un péché. Ainsi, les péchés les plus graves sont l&rsquo;hérésie, le polythéisme, la fornication, l&rsquo;apostasie, les jeux de hasard, etc., tout ce qui est haram (interdit).</p>



<p>Le musulman utilise souvent l&rsquo;expression &laquo;&nbsp;<em>inch Allah&nbsp;&raquo;, </em>&laquo;&nbsp;si Dieu le veut&nbsp;&raquo;. Ce que corrobore le Coran (18, 23-24) :</p>



<blockquote class="wp-block-quote is-layout-flow wp-block-quote-is-layout-flow"><p>Et ne dis jamais à propos d&rsquo;une chose : &laquo;&nbsp;Je le ferai sûrement demain&nbsp;&raquo; sans ajouter &laquo;&nbsp;si Allah le veut&nbsp;&raquo;, et invoque ton Seigneur quand tu oublies et dis : &laquo;&nbsp;Je souhaite que mon seigneur me guide et me mène plus près de ce qui est correct&nbsp;&raquo;.<br></p></blockquote>



<p>On pourrait donc croire que toutes les actions sont déterminées par la volonté d&rsquo;Allah. D&rsquo;autant plus qu&rsquo;à la sourate 8, 17, on lit : &laquo;&nbsp;<em>Ce n&rsquo;est pas vous qui les avez tué, mais c&rsquo;est Allah qui les a tués&#8230; Allah est audient (NB : il peut entendre  tout) et omniscient.</em>&laquo;&nbsp;</p>



<p>Or dans l&rsquo;islam, comme dans tous les courants philosophiques, les oulémas débattent pour savoir si l&rsquo;homme a son libre arbitre ou s&rsquo;il est prédestiné. Ainsi &laquo;&nbsp;<em>tout ce qui est généré par nos actes est notre action&nbsp;&raquo; </em>s&rsquo;oppose à &laquo;&nbsp;<em>l&rsquo;homme n&rsquo;a qu&rsquo;une connaissance partielle des effets de ses actes&nbsp;&raquo;.</em> Entre ces deux extrêmes, on trouve : &laquo;&nbsp;<em>si l&rsquo;homme connaît la modalité de ses actes, il a le libre arbitre, sinon, l&rsquo;acte doit être attribué à Allah</em>&laquo;&nbsp;.</p>



<p>Ce débat était déjà présent en Mésopotamie, bien avant l&rsquo;islam. Ainsi un texte s&rsquo;interroge : &laquo;&nbsp;<em>Pourquoi être irréprochable dans son comportement personnel, social et religieux si l&rsquo;on peut être puni pour une faute dont on n&rsquo;a même pas conscience ? A quoi servent les bonnes actions si elles ne garantissent pas une vie sans épreuves ?&nbsp;&raquo;</em></p>



<h5 class="wp-block-heading">Le péché originel</h5>



<p>C&rsquo;est le nom que donnent les chrétiens à la cause du renvoi d&rsquo;Adam et de Ève du Paradis dans le roman de la Bible.<br>Pour rappel, Adam a été créé à l<em>&lsquo;image de Dieu</em>. Pour le sortir de sa solitude, Dieu lui a façonné une compagne, Ève. Adam est libre dans le Paradis, une seule interdiction lui a été faite : ne pas &laquo;&nbsp;<em>manger</em>&nbsp;&raquo; de l&rsquo;arbre de la connaissance du bien et du mal. Le serpent tente Ève qui pousse son compagnon à manger le fruit de cet arbre. L&rsquo;exclusion du Paradis sera le châtiment pour cette désobéissance.</p>



<p>Dans l’Épître aux Romains 5, 12, attribuée à Paul, on lit &laquo;&nbsp;<em>&#8230; de même que par un seul homme (Adam), le péché est entré dans le monde</em>, <em>et <strong>par le péché, la mort</strong>, et qu&rsquo;ainsi la mort a passé en tous les hommes, situation dans laquelle tous ont péché</em>&laquo;&nbsp;.</p>



<p>Paul lie le péché originel à la mort. Or dans la Bible, rien ne dit qu&rsquo;Adam et Ève sont immortels, au contraire. Lorsqu&rsquo;il constate la désobéissance, Dieu dit &laquo;&nbsp;<em>Voilà que l&rsquo;homme est devenu comme l&rsquo;un de nous, pour connaître le bien et le mal. Qu&rsquo;il n&rsquo;étende pas maintenant la main sur l&rsquo;arbre de vie, n&rsquo;en mange et <strong>ne vive pour toujours</strong>&nbsp;&raquo; (Gen. 3, 22)</em>. Ce personnage énigmatique qu&rsquo;est Paul (j&rsquo;y reviendrai dans un prochain article) ne connaît pas la Torah alors qu&rsquo;il a la prétention de diriger la vie spirituelle des premiers chrétiens.</p>



<p>D&rsquo;après le raisonnement dogmatique de Paul, basé sur un récit mythologique, l&rsquo;homme est souillé dès sa naissance. Il porte la responsabilité de la faute d&rsquo;Adam qui lui est transmise par hérédité. Augustin d&rsquo;Hippone reprend ce discours : l&rsquo;homme est souillé par engendrement. Le baptême efface la souillure. Or comme on baptise les enfants, c&rsquo;est la preuve qu&rsquo;ils sont souillés dès la naissance. Le péché originel est un péché de chair. Pour lui, c&rsquo;est sexuel !</p>



<p>Seuls les catholiques adhérent à ce dogme. Les protestants et les orthodoxes s&rsquo;en sont éloignés.<br>Les juifs ignorent la conséquence de l&rsquo;acte d&rsquo;Adam, ils sont les élus de Dieu.<br>Les musulmans ont une interprétation différente (Co. 20, 115-123) :</p>



<blockquote class="wp-block-quote is-layout-flow wp-block-quote-is-layout-flow"><p>Nous avons auparavant fait une recommandation à Adam : mais il l&rsquo;oublia et nous n&rsquo;avons pas trouvé chez lui de résolution ferme.<br>Et quand nous dîmes aux anges de se prosterner devant Adam, ils se prosternèrent excepté Iblis (le Diable) qui refusa.<br>Alors nous dîmes : &laquo;&nbsp;Ô Adam, celui-là est vraiment un ennemi pour toi et ton épouse (NB : elle n&rsquo;a pas de nom). Prenez garde qu&rsquo;il vous fasse sortir du Paradis, car alors tu seras malheureux.<br>Car tu n&rsquo;y auras pas faim, ni ne sera nu.<br>Tu n&rsquo;y auras pas soif ni seras frappé par l&rsquo;ardeur du soleil.<br>Puis le Diable le tenta en disant : Ô Adam, t&rsquo;indiquerai-je l&rsquo;arbre de l&rsquo;éternité et un royaume impérissable ?&nbsp;&raquo;<br>Tous deux en mangèrent. Alors leur apparut leur nudité. Ils se mirent à se couvrir avec des feuilles du paradis. Adam désobéit ainsi à son Seigneur et il s&rsquo;égara.<br>Son Seigneur l&rsquo;a ensuite élu, agréé son repentir et l&rsquo;a guidé.<br>Il dit : &laquo;&nbsp;Descendez d&rsquo;ici (NB : le paradis n&rsquo;est pas sur terre). Vous serez tous ennemis les uns des autres (Note du Coran de Médine : on peut comprendre les humains et les diables). Puis si jamais un guide vous vient de ma part, quiconque suit mon guide ne s&rsquo;égarera ni ne sera malheureux.<br></p></blockquote>



<h5 class="wp-block-heading">Péché et rédemption dans le christianisme</h5>



<p>Pour le christianisme, l&rsquo;homme est un pécheur et réside, jusqu&rsquo;à sa mort, dans un monde soumis au Diable. Dans les évangiles, la plupart des miracles de Jésus sont des exorcismes, il chasse des démons.<br>Le péché est le fonds de commerce du catholicisme. La confession mensuelle était une obligation il n&rsquo;y a pas encore longtemps et le prêtre était habilité à pardonner (ou faire pardonner) les péchés des paroissiens.<br>Je reviens sur la définition chrétienne de rédemption : &laquo;&nbsp;<em>l&rsquo;acte par lequel Jésus rachète les hommes esclaves de leurs péchés en le payant de sa vie.</em>&nbsp;&raquo; Elle amène plusieurs questions. De quoi Jésus sauve-t-il ? En quoi sa mort sauve-t-elle ? En quoi la crucifixion est-elle nécessaire à notre rachat ? Pour répondre à ces questions, j&rsquo;ai consulté le courrier des lecteurs du quotidien catholique français <em>La Croix</em> espérant y trouver des réponses concrètes. Désillusion! Je n&rsquo;ai trouvé que des lieux communs et des phrases vides de sens. Il faut dire que le sujet est délicat.</p>



<p>Florilège.<br><strong>En quoi le Christ nous sauve-t-il ?</strong><br>Le rédacteur prend d&rsquo;abord une précaution : &laquo;&nbsp;<em>Dès que l&rsquo;on affirme que le Christ nous apporte le salut, les choses sont nettement moins claires</em>&laquo;&nbsp;. Un théologien se porte à son secours (je résume) : Nous souhaitons tous le bonheur, aimer et être aimé, mais nous ne pouvons l&rsquo;atteindre par nos propres moyens. Les chrétiens sont aimés par Dieu, gratuitement, c&rsquo;est cela le salut que l&rsquo;homme reçoit. <em>Nous ne sommes pas sauvé par la mort de Jésus, mais par son amour</em>.<br>Par sa résurrection, Jésus donne l&rsquo;exemple, elle ne sera pas la seule. Si le salut est offert à tous, il n&rsquo;est pas automatique : il faut avoir la foi. Jésus ne te sauvera pas sans toi.</p>



<p><strong>Quel lien entre la mort du Christ et nos péchés ?</strong><br>Dans la réponse précédente, on nous disait : <em>Nous ne sommes pas sauvé par la mort de Jésus, mais par son amour</em>. Qu&rsquo;en est-il ici ?<br>Il faut revenir à des vérités simples : Jésus est mort et ressuscité pour nous les hommes, pour notre salut. <br>C&rsquo;est tout ce qu&rsquo;on peut tirer de la réponse.</p>



<p><strong>La croix est-elle nécessaire à notre rachat ?</strong><br>La réponse est un coupé-collé de la question précédente.<br>C&rsquo;est le père jésuite Michel Souchon qui repond(ait). Il conclut par &laquo;&nbsp;<em>Jésus nous a révélé le visage du Père, un dieu à visage humain qui veut aimer l&rsquo;homme et non pas le faire payer. Révélation formidable pour nous les hommes et pour notre salut. Pour cette révélation, Jésus a mis en jeu sa propre vie, il nous a libéré du péché au prix de sa vie&nbsp;&raquo;.</em></p>



<p>Je suis désolé de vous avoir laissé sur votre faim, mais je ne peux pas faire mieux que des théologiens aguerris. Leur raisonnement est une boucle : salut &#8211; mort &#8211; résurrection. Au IIe siècle de notre ère, une des innombrables sectes chrétiennes, les gnostiques, ne croyait pas à la crucifixion, ni à la résurrection de Jésus. Pour les gnostiques, c&rsquo;est l&rsquo;enseignement de Jésus qui indiquait le chemin du salut.</p>



<p>Qu&rsquo;est-ce qui a changé, sur terre, depuis la période de Jésus ? Rien, strictement rien. J&rsquo;ai peine à imaginer que Dieu ait construit un complexe hôtelier tout neuf, appelé Paradis, pour accueillir les âmes sauvées par Jésus.</p>

<script>
(function() {
	'use strict';

	const fetches = {};
	const promises = {};
	const urls = {
		'wp-polyfill': '/wp-includes/js/dist/vendor/wp-polyfill.min.js?m=1764669441i&ver=3.15.0',
		'verbum': '/wp-content/mu-plugins/jetpack-mu-wpcom-plugin/sun/jetpack_vendor/automattic/jetpack-mu-wpcom/src/build/verbum-comments/verbum-comments.js?m=1765400473i&minify=false&ver=87a45c8962a448c74c1e'
	};
	const loaders = {
		'verbum': () => {
			fetchExternalScript('wp-polyfill');
			fetchExternalScript('verbum');
			promises['wp-polyfill'] = promises['wp-polyfill'] || loadWPScript('wp-polyfill');
			promises['verbum'] = promises['verbum'] || promises['wp-polyfill'].then( () => loadWPScript('verbum') );
			return promises['verbum'];
		},
		
	};
	const scriptExtras = {
		'verbum': { translations: 1, before: 0, after: 0 },
		
	};

	window.WP_Enqueue_Dynamic_Script = {
		loadScript: (handle) => {
			if (!loaders[handle]) {
				console.error('WP_Enqueue_Dynamic_Script: unregistered script `' + handle + '`.');
			}
			return loaders[handle]();
		}
	};

	function fetchExternalScript(handle) {
		if (!urls[handle]) {
			return Promise.resolve();
		}

		fetches[handle] = fetches[handle] || fetch(urls[handle], { mode: 'no-cors' });
		return fetches[handle];
	}

	function runExtraScript(handle, type, index) {
		const id = 'wp-enqueue-dynamic-script:' + handle + ':' + type + ':' + (index + 1);
		const template = document.getElementById(id);
		if (!template) {
			return Promise.reject();
		}

		const script = document.createElement( 'script' );
		script.innerHTML = template.innerHTML;
		document.body.appendChild( script );
		return Promise.resolve();
	}

	function loadExternalScript(handle) {
		if (!urls[handle]) {
			return Promise.resolve();
		}

		return fetches[handle].then(() => {
			return new Promise((resolve, reject) => {
				const script = document.createElement('script');
				script.onload = () => resolve();
				script.onerror = (e) => reject(e);
				script.src = urls[handle];
				document.body.appendChild(script);
			});
		});
	}

	function loadExtra(handle, pos) {
		const count = (scriptExtras[handle] && scriptExtras[handle][pos]) || 0;
		let promise = Promise.resolve();

		for (let i = 0; i < count; i++) {
			promise = promise.then(() => runExtraScript(handle, pos, i));
		}

		return promise;
	}

	function loadWPScript(handle) {
		// Core loads scripts in this order. See: https://github.com/WordPress/WordPress/blob/a59eb9d39c4fcba834b70c9e8dfd64feeec10ba6/wp-includes/class-wp-scripts.php#L428.
		return loadExtra(handle, 'translations')
			.then(() => loadExtra(handle, 'before'))
			.then(() => loadExternalScript(handle))
			.then(() => loadExtra(handle, 'after'));
	}
} )();
