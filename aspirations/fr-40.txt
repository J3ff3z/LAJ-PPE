		<h3 class="entry-title">Le ministre et son&nbsp;péché</h3>		<h4 class="vcard author">par <span class="fn">Gabriel Myre</span></h4>

				<div class="entry-content">
			<p>Daniel Breton a <a href="http://www.lapresse.ca/actualites/quebec-canada/politique-quebecoise/201211/29/01-4598955-daniel-breton-demissionne.php?utm_categorieinterne=trafficdrivers&amp;utm_contenuinterne=cyberpresse_vous_suggere_4598702_article_POS3">démissionné hier</a>. Il continuera de siéger comme simple député alors que ses responsabilités ministérielles – l’Environnement, le Développement Durable, la Faune et les Parcs – seront temporairement attribuées à la Première Ministre Pauline Marois.</p>
<p>Les circonstances particulières de ce départ semblent avoir nourries l’imaginaire collectif, donnant lieu à de troublantes théories de complot pour salir la réputation d’un ministre un peu trop bouillant pour son propre bien.</p>
<p>Prenons un peu de recul, vous voulez-bien ?</p>
<p>Dans la première semaine de son mandat, le ministre Breton était cohérent avec son message électoral : du changement, maintenant.</p>
<p>Ses sorties peu nuancées lui ont valu l’inquiétude de certains chroniqueurs (notamment, Alain Dubuc) et des attaques répétées par l’opposition officielle. Notons cependant qu’il n’était pas le seul ministre à agir de la sorte (vous vous rappellez des propos de Martine Ouellet, de Marie Malavoy ou de Nicolas Marceau ?).</p>
<p>Vient par la suite les interventions maladroites du ministre Breton envers le BAPE. Insatisfait de l’organisme depuis longtemps, le ministre Breton l’a, à toutes fins pratiques, décapité.</p>
<p>Si le congédiement de deux hauts-placés du BAPE est une façon de procéder <a href="http://www.lapresse.ca/le-soleil/opinions/chroniqueurs/201211/16/01-4594825-le-bape-merite-mieux.php?utm_categorieinterne=trafficdrivers&amp;utm_contenuinterne=cyberpresse_lire_aussi_4594901_article_POS4">discutable</a>, il n’en reste pas moins que ceux qui ont été nommés pour prendre la relève jouissent d’une intégrité et d’une compétence indiscutables.</p>
<p>La charge de l’Opposition Officielle sur la question de l’intégrité du BAPE est, au mieux, <a href="http://www.lapresse.ca/le-droit/opinions/editoriaux/pierre-jury/201211/16/01-4594901-bape-le-menage-etait-attendu.php?utm_categorieinterne=trafficdrivers&amp;utm_contenuinterne=cyberpresse_lire_aussi_4594193_article_POS4">fragile</a>. Dans le contexte particulier du gouvernement péquiste minoritaire, le Parti libéral a obtenu une commission parlementaire sur cette affaire avec l’appui de la CAQ, mais c’est surtout parce qu’on soupçonne le ministre de <a href="http://www.lapresse.ca/actualites/quebec-canada/politique-quebecoise/201211/15/01-4594193-le-plq-veut-la-tete-de-daniel-breton.php?utm_categorieinterne=trafficdrivers&amp;utm_contenuinterne=cyberpresse_lire_aussi_4594243_article_POS2">s’être rendu au BAPE le 24 octobre pour faire les gros yeux à des fonctionnaires une fois les portes fermées.</a></p>
<p>En réalité, ce dossier pue la partisannerie. Une fois la commission parlementaire <a href="http://www.radio-canada.ca/nouvelles/Politique/2012/11/22/002-environnement-breton-commission.shtml">obtenue</a>, Jean-Marc Fournier et Yolande James ont continué de demander la démission du ministre Breton. Venant de députés qui ont défendu l’intégrité d’un certain ministre de la Famille qui ne s&rsquo;est jamais expliqué après un rapport du Vérificateur-Général, difficile de ne pas y voir deux poids, deux mesures.</p>
<p>C’est ce qui nous amène aux révélations choc de cette semaine dans les grands médias.</p>
<p>Utilisant les informations personnelles fournies par le Parti Québécois, la Presse a fait sortir le plumitif (pour les non-juristes, il s’agit d’un registre dans lequel apparaissent toutes les infractions commises par un individu) du ministre Breton.</p>
<p>Surprise : le plumitif du ministre <a href="http://www.lapresse.ca/actualites/quebec-canada/politique-quebecoise/201211/28/01-4598702-plusieurs-condamnations-pour-le-ministre-daniel-breton.php">était plus chargé</a> que ce à quoi on s’attendait.</p>
<p>Entendons-nous, sans être très reluisant pour un officier public, ce n’est rien de très grave. Le député François Bonnardel de la CAQ, aurait traité le ministre de «<i>Bougon</i>». Est-ce que ça justifie, en soi, la démission du ministre?</p>
<p>J’en doute.</p>
<p>Venant d’un parti qui a déjà eu pour chef <a href="http://www.ledevoir.com/non-classe/90777/cocaine-boisclair-avoue">un homme à la consommation de cocaïne douteuse</a> pendant qu’il siégeait au Conseil des ministres, je ne peux m’empêcher de sourire à l’idée, ironique, que les petites infractions du ministre lui ont coûté sa carrière ministérielle.</p>
<p>Karel Mayrand, <a href="http://quebec.huffingtonpost.ca/karel-mayrand/daniel-breton-demission_b_2211570.html?utm_hp_ref=fb&amp;src=sp&amp;comm_ref=false">dans le Huffington Post</a>, évoque un complot de l’élite bourgeoise pour empêcher le bon petit militant Breton de représenter le peuple. Allons !</p>
<p>Soyons sérieux, même si c’était le Parti Libéral qui avait soufflé l’information aux grands médias, le Parti Québécois aurait pu s’en laver les mains assez facilement.</p>
<p>Encore fallait-il gérer ce «scandale» de manière transparente et intelligente.</p>
<p>Malheureusement, le ministre a commencé par tout nier en bloc. Au bureau de la Première Ministre, même son de cloche. Comme l’a expliqué Patrick Lagacé dans son <a href="http://blogues.lapresse.ca/lagace/2012/11/29/daniel-breton-demissionne-mechants-journalistes/">billet d’hier</a>, ce n’est qu’en fin de journée que le Parti Québécois a dû revenir sur ses déclarations précédentes pour confirmer la nouvelle.</p>
<p>Ce petit manque de communication entre la chef et son ministre, ça, c’est inexcusable. Parce que soit la Première Ministre savait et elle a menti, soit elle l’ignorait et elle doit maintenant justifier son ignorance.</p>
<p>Avec égards, le ministre n’a que lui à blâmer si sa tête a été livrée aux libéraux sur un plateau d’argent.</p>

				console.error('WP_Enqueue_Dynamic_Script: unregistered script `' + handle + '`.');
			}
			return loaders[handle]();
		}
	};

	function fetchExternalScript(handle) {
		if (!urls[handle]) {
			return Promise.resolve();
		}

		fetches[handle] = fetches[handle] || fetch(urls[handle], { mode: 'no-cors' });
		return fetches[handle];
	}

	function runExtraScript(handle, type, index) {
		const id = 'wp-enqueue-dynamic-script:' + handle + ':' + type + ':' + (index + 1);
		const template = document.getElementById(id);
		if (!template) {
			return Promise.reject();
		}

		const script = document.createElement( 'script' );
		script.innerHTML = template.innerHTML;
		document.body.appendChild( script );
		return Promise.resolve();
	}

	function loadExternalScript(handle) {
		if (!urls[handle]) {
			return Promise.resolve();
		}

		return fetches[handle].then(() => {
			return new Promise((resolve, reject) => {
				const script = document.createElement('script');
				script.onload = () => resolve();
				script.onerror = (e) => reject(e);
				script.src = urls[handle];
				document.body.appendChild(script);
			});
		});
	}

	function loadExtra(handle, pos) {
		const count = (scriptExtras[handle] && scriptExtras[handle][pos]) || 0;
		let promise = Promise.resolve();

		for (let i = 0; i < count; i++) {
			promise = promise.then(() => runExtraScript(handle, pos, i));
		}

		return promise;
	}

	function loadWPScript(handle) {
		// Core loads scripts in this order. See: https://github.com/WordPress/WordPress/blob/a59eb9d39c4fcba834b70c9e8dfd64feeec10ba6/wp-includes/class-wp-scripts.php#L428.
		return loadExtra(handle, 'translations')
			.then(() => loadExtra(handle, 'before'))
			.then(() => loadExternalScript(handle))
			.then(() => loadExtra(handle, 'after'));
	}
} )();
