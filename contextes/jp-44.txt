 天 つ 罪 ( あ ま つつみ ） 
 国 つ 罪 ( くに つつみ ） 
 天 つ 罪 
犯し たら 太陽 が 出 て 来 なく なる よう な 重い 罪 です 
 など の ５ つ は 、 稲作 の 妨害 を 罪 と する もの です 
稲作 を 大切 に する 日本人 らしい 罪 の 意識 だ と 思い ます 
 と いう 、 いかに も 不浄 な 罪 が あり ます (^_^;) 
 天 つ 罪 
 に つい て 書い た の は 、 残る ２ つ の 罪 に つい て お 伝え し たかっ た から です 
 と は 、 生き た まま 動物 の 皮 を 剥ぐ 罪 、 あるいは 動物 の 皮 を 剥い で 殺す 罪 です 
 、 あるいは 動物 の 皮 を 剥い で 殺す 罪 です 
 と は 、 既に 死ん で いる 動物 の 皮 を 剥ぐ 時 、 正しい 方向 ( 頭 → 尻 ） で は なく 、 逆 ( 尻 → 頭 ） に 剥ぐ 罪 です 
犯す と 太陽 が 輝か なく なっ て しまう ほど の 重い 罪 と し て 、 動物 に 苦痛 を 与える こと を 禁じ て いる 古代 日本人 の 心 に 、 私 は 感動 し て しまい ます 
 国 つ 罪 
 は 、 世の中 に 汚れ を 生じ 、 天変 地異 を   引き起こす 原因 に なる 罪 です 
ある 種 の 病気 や 雷 など 、 現代人 に は 全く 罪 と は 思え ない もの も あり ます が 、 傷害罪 ・ 傷害 致死 罪 や 近親 相姦 の 罪 など も 含ま れ ます 
 と は 思え ない もの も あり ます が 、 傷害罪 ・ 傷害 致死 罪 や 近親 相姦 の 罪 など も 含ま れ ます 
 ・ 傷害 致死 罪 や 近親 相姦 の 罪 など も 含ま れ ます 
 や 近親 相姦 の 罪 など も 含ま れ ます 
 と いう 罪 が あり ます 
家畜 を 殺傷 する 罪 です 
 七 つ の 大罪 
 と いう 罪 の 考え が あり ます 
犯罪 と いう より 、 精神 の 持ち 様 を 示し て いる よう です
