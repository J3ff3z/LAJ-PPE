です 。 つまり 、 罪 の 根源 、 悪い 行為 や 悪い 考え の 「 指導者 」 で ある と 言え ます 。   「 首都 」 と いう 用語 は 、 「 頭 」 、 「 上部 」 を 意味 する ラテン語 の   caput 

に 由来 し て い ます 。 



７ つ の 罪 と は 、   高慢   、   強欲   、   妬み   、   怒り   、   色欲   、   暴食   、   怠惰   です 。 



1 .   素晴らしい 



プライド は   過度 の プライド 

と し て 定義 でき ます 。 それ は 、 自分 が 他 の 人 より も 優れ て いる と 考える 傾向 です 。 プライド は 、 あたかも 自分 が すべて や 誰 より も 優れ て いる か の よう に 考え 、 行動 する 、 非常 

に 虚栄心 の ある 人 の 罪 です 。 誇り の 反対 は 謙虚 さ です 。 



カトリック 教 徒 に とっ て 、 プライド は 創世記 に 記述 さ れ て いる 原罪 の 一部 で ある ため 、 主な 罪 、 また は すべて の 罪 の 根源 です 。 



神 は アダム と イブ が 善悪 の 知識 の 木の実 を 味わう こと を 禁じ まし た 。 しかし 、 悪魔 は 蛇 の 姿 で 二人 を 誘惑 し 、 その 実 を 味わえ ば 神 の よう に 善悪 を 知る よう に なる だろう と 言い 

まし た 。 アダム と エバ は 主 から 独立 し たい と 願い 、 不 従順 と 高慢 の 罪 を 犯し まし た 。 



高慢 の 罪 に つい て 言及 し て いる 聖書 の 一節 に 次 の よう な もの が あり ます 。 



主 は 心 の 中 で 高ぶる 者 を 憎ま れ ます 。 彼 ら は 間違い なく 処罰 さ れる だろう 。

2 .   強欲 



貪欲 と は 、 貪欲 と も 呼ば れ 、   物質的 な 物 や お金 に 対する 過剰 な 執着 の こと 

です 。 貪欲 な 人 は ケチ です 。 つまり 、 自分 が 持っ て いる もの を 共有 する の が 好き で は なく 、 常 に もっと 持つ ため に あらゆる こと を し ます 。 強欲 は 寛大 さ の 反対 です 。 



聖書 に は 貪欲 に つい て ほのめかし た 箇所 が たくさん あり ます 。 その 一 つ は 、 使徒 パウロ が 書い た テモテ へ の 第 一 の 手紙 の 中 に あり ます 。 



お金 へ の 愛 は すべて の 悪 の 根源 で ある ため 、 金持ち に なり たい 人 は 誘惑 、 罠 、 そして 多く の 制御 不能 で 有害 な 欲望 に 陥り 、 それ が 人々 を 破滅 と 破壊 に 陥ら せ ます 。 お金 を 欲し 

がっ た ため に 信仰 から 外れ 、 多く の 苦しみ を 経験 し た 人 も い ます 。

3 .   羨望 



羨望 は   他人 の 利益 の ため の 悲しみ 

です 。 嫉妬 深い 人 は 、 他人 の 成果 を 不快 に 感じ 、 他人 の 勝利 が 自分 の 損失 で ある か の よう に 他人 を 幸せ に する こと が でき ない 人 です 。 羨望 の 反対 は 、 慈善 、 無 執着 、 利他 主義 

です 。 



聖書 に は ねたみ に 関する 記述 が たくさん あり ます 。 最も 重要 な もの の   1   つ は 、 最初 の 殺人 に つい て 語る もの です   ( 創世記   4

悪徳 は 嫉妬 です 。

5 .   欲望 



情欲 、 好色 、 放蕩 は   性的 欲求 に 関連 し た 罪   です 。 カトリック 教 徒 に とっ て 、 この 罪 は 性的 虐待 や 性的 快楽 の 過剰 な 追求 に 関連 し て い ます 。 欲望 の 反対 は 純粋 さ です 。 



この 罪 を 扱っ た 聖書 の 一節 の 中 で 、 最も 鋭敏 な もの の 一 つ は ガラテヤ   5   章   19   節 です 。

6 .   大食い 



暴食 は 、 自分 の 必要 を 超え て   過度 に 食べ たり 飲ん だり する 欲求 に 関連 する 罪 

です 。 この 罪 は 、 自分 の 体 を コントロール でき なく なる こと と 関係 し て い ます 。 暴食 の 反対 は 節制 です 。 



実際 、 ほとんど すべて の 罪 は 節度 の 欠如 に 関係 し て い ます 。 暴食 の 場合 は 、 食べ物 や 飲み物 の 過剰 摂取 で あり 、 怠惰 など の 他 の 罪 に つながる 可能性 が ある ため 、 肉体的 および 

精神的 な 悪 に 起因 する と 考え られ て い ます 。 暴食 は 、 物質的 な もの に 幸福 を 求める こと の 現れ です 。 



箴言   23 章 20 節 から 21 節 に は ， 暴食 の 誘惑 から 遠ざかり たい 人 へ の アドバイス が あり ます 。 



ワイン に 酔っ て いる 人 や 肉 を むさぼり食べる 人 たち と は 付き合っ て は いけ ませ ん 。 大 酒飲み や 大食漢 は 貧乏 に なり 、 眠気 で ボロ布 を まとう こと に なる から で ある 。

7 .   怠惰 



怠惰 と は 、 身体的 で あれ 知的 で あれ 、   ある 程度 の 努力 を 必要 と する 活動 に 対する 意志 や 関心 の 欠如 

です 。 これ は 、 行動 の 欠如 、 仕事 や その 他 の 日常 業務 に 対する 熱意 の 欠如 と し て 定義 でき ます 。 怠惰 の 反対 は 努力 、 意志力 、 行動 です 。 



カトリック の 信奉 者 に とっ て 、 怠惰 の 罪 は 、 （ 毎日 の 糧 を 探す と いう ） 仕事 の 義務 を 自発的 に 拒否 する こと と 関係 が ある が 、 献身 と 美徳 の 探求 の 実践 に おけ る 勇気 の 欠如 と も 

関係 し て いる 。 



この 罪 に つい て 語ら れ て いる 聖書 の 一節 の 中 で 、 次 の よう な 箇所 が 際立っ て い ます 。 



怠け者 は 欲望 し て も 何 も 持た ない が 、 努力 する 者 の 欲望 は 叶え られる 。

七 つ の 罪 の 起源 は 、 キリスト教 の 僧侶 エヴァグリウス ・ ポンティコス （ 西暦 345 年 ～ 399 年 ） が 、 日常的 な 宗教 習慣 を 混乱 さ せる 主な 悪い 考え を 列挙 する ため に 書い た リ 

スト に あり ます 。 リスト は 7 つ の 罪 で は なく 、 8 つ の 罪 で 構成 さ れ て い まし た 。 現在 知ら れ て いる もの に 加え て 、 悲しみ が あり まし た 。 そこ に は 羨望 は なく 、 虚栄心 （ 虚栄心 

） が あっ た 。

に よっ て ラテン語 に 翻訳 さ れ 書き直さ れ まし た 。 彼 は 怠惰 を 排除 し 、 嫉妬 を 加え 、 主な 罪 と し て プライド を 選び まし た 。 



13   世紀 に 修道士 トーマス   アクィナス   ( ) 

が リスト を 復元 し 、 悲しみ の 代わり に 怠惰 が 再び 含ま れ まし た 。 今日 私たち が 知っ て いる 「 七 つ の 大罪 」 は 、 トマス ・ アクィナス の リスト に 遡り ます 。 



聖書 の テーマ に 関連 し て いる に も かかわら ず 、 七 つ の 大罪 は 聖書 に 名前 も 記載 さ れ て い ませ ん 。 それ ら は 後 に カトリック 教会 に よっ て 作成 さ れ まし た 。 



しかし 、 人々 の 生活 に おけ る 罪 の 起源 を 宗教的 に 説明 する と どう なる でしょう か ? マルコ   7 : 21 - 23   に は 、 この 問題 を 明らか に する 一節 が あり ます 。
