今回 は いきなり 重たい タイトル です 。 こう いう タイトル の 漫画 も あり ます ね 。 



７ つ の 大罪 ( mortal 

sins ) と いう 理解 の 仕方 自体 は 、 カトリック 教会 の もの で 、 僕 は あまり 馴染み が あり ませ ん でし た 。 しかし 、 ある 投稿 が きっかけ と なり 、 興味 を 持っ た の です 。 



７ つ の 大罪 と いう から に は よほど 凄い 罪 な ん だろう な と 思っ て 調べ て み たら 、 何 と 自分 に も 大いに 当てはまる こと ばかり で は ない か 、 と 少し びっくり し まし た 。 



七 つ の 大罪 と は 、 傲慢 （ Pride ） 、 嫉妬 （ Envy ） 、 憤怒 （ Wrath ） 、 怠惰 （ Sloth ） 、 強欲 （ Greed ） 、 暴食 （ Gluttony ） 、 色欲 （ Lust ） 

の 7 つ の 罪 を 指し ます 。 ﻿ 



これ ら の リスト を 見 て 、 自分 は 関係 ない よ 、 と 言える 人 は 一人 も い ない の で は ない でしょう か 。 



むしろ 私たち の 社会 は これ ら の 「 大罪 」 に 満ち て おり 、 そこ から あらゆる 問題 が 発生 し ます 。 残念 ながら 、 この インターネット 空間 も これ ら に 満ち て い て 、 知ら ず に 多く の 人 

が 巻き込ま れ て しまい ます 。 



僕 自身 も 静か に 考え て みる と 、 思い当たる 場面 が ありあり と 目 に 浮かび ます 。 



さて 、 人 は これ ら の ７ つ の 大罪 を 克服 できる の でしょう か 。 



新約 聖書 に 収め られ て いる 多く の 書簡 を 書い た パウロ は 、 ローマ人 へ の 手紙 の 中 で 興味 深い こと を 書い て い ます 。 



私 は 、 自分 で し たい と 思う 善 を 行わ ない で 、 かえって し たく ない 悪 を 行っ て い ます 。 もし 私 が 自分 で し たく ない こと を し て いる の で あれ ば 、 それ を 行っ て いる の は 、 もはや 私 

で は なく て 、 私 の うち に 住む 罪 です 。 そう いう わけ で 、 私 は 、 善 を し たい と 願っ て いる の です が 、 その 私 に 悪 が 宿っ て いる と いう 原理 を 見いだす の です 。 



ローマ人 へ の 手紙 7 章 19 - 21 節 



つまり 、 人間 の 本質的 部分 その もの に 「 罪 」 が 深く 巣食っ て いる と いう の です 。 



な の で 、 「 7 つ の 大罪 」 を 克服 せ ん と する 人間 の 努力 自体 が 徒労 に 終わる と 言っ て いる の です 。 



で は 、 私たち は 開き直っ て 、 罪 を 犯す べき な の でしょう か 。 



そんな こと は 全く ない の です 。 パウロ は きちんと 解決 の 道 を 示し て い ます 。 



なぜ なら 、 キリスト ・ イエス に ある 、 いのち の 御 霊 の 原理 が 、 罪 と 死 の 原理 から 、 あなた を 解放 し た から です 。 



ローマ人 へ の 手紙 8 章 2 節 



キリスト ・ イエス に ある 、 いのち の 御 霊 の 原理 が 、 人間 の 根本的 な 部分 に 働きかける の です 。 



鍵 は 、 キリスト ・ イエス を 心 に 受け入れ た 者 で あれ ば 、 キリスト と とも に 十字架 に つけ られ た 、 と いう 事実 を 受け入れる こと です 。 「 克服 する 」 の で は なく 、 「 死ぬ 」 の です 

。 



私 は キリスト と とも に 十字架 に つけ られ まし た 。 もはや 私 が 生き て いる の で は なく 、 キリスト が 私 の うち に 生き て おら れる の です 。

もし 、 私たち が 自分 の 罪 を 言い表す なら 、 神 は 真実 で 正しい 方 です から 、 その 罪 を 赦し 、 すべて の 悪 から 私たち を きよめ て ください ます 。
