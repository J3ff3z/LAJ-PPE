@ Rico です 。 



日々 を 懸命 に 生きる 中 で 、 ふと 「 これ で いい の だろう か 」 と 立ち止まっ たり 、 知ら ず 知ら ず の うち に 誰 か を 傷つけ 、 自分 自身 を 責め て しまっ たり する こと は ない でしょう か 。 

われわれ が 生きる 現代 社会 は 、 複雑 で 、 時 に 息苦し さ を 感じる こと も 少なく あり ませ ん 。 



そんな 今 だ から こそ 、 千 年 以上 も の 間 、 日本 の 精神性 の 根幹 を 成し て き た 「 大 祓詞 （ お お はらえ の ことば ） 」 に 、 生きる 本質 を 知る ヒント が ある の で は ない か と 思い 、 今回 は 

「 大 祓詞 」 に 示さ れる 内容 、 そこ で 記さ れる 「 罪（ツミ） 」 に つい て 深 掘り し て いき たい と 思い ます 。 



この 祝詞 に は 、 「 天 つ 罪 」 と 「 国 つ 罪 」 と いう 、 古代 日本人 が 捉え た 「 罪 」 の 形 が 記さ れ て い ます 。 それ は 、 現代 で いう 単なる 法律 違反 の こと のみ に 留まり ませ ん 。 われわれ 

の 心身 や 人間 関係 、 さらに は 自然 と の 関わり に まで 及ぶ 、 根源 的 な 「 過ち 」 や 「 穢れ 」 を 指し示す もの です 。 

さらに 、 江戸 時代 の 国学者 ・ 本居 宣長 が 、 その 鋭い 洞察力 で 解き明かし た 「 ツミ 」 の 言わ ん と し て いる こと 、 その 真相 を 紐解い て いき ます 。 



宣長 の 解釈 を 知る こと で 、 われわれ が 抱き がち な 「 罪悪感 」 と いう 重荷 から 解放 さ れ 、 人生 を リセット し 、 前向き に 再出発 する ため の 新た な 視点 が 得 られる はず です 。 

古代 の 叡智 と 近世 の 知性 が 交差 する 地点 から 、 人生 を より 深く 、 豊か に 生きる ため の 本質 を 探求 し て いき ましょう 。 



（ 大 祓詞 シリーズ ： 以下 も 参照 ください ） 



1 . 大 祓詞 が 示す 「 罪 」 の 全体像   ～ 天 つ 罪 と 国 つ 罪 ～ 



まず 、 大 祓詞 が どのような 「 罪 」 を 問題 に し て いる の か 、 その 全体像 を 把握 する こと から 始め ます 。 大 祓詞 に は 、 大きく 分け て 「 天 つ 罪 」 と 「 国 つ 罪 」 の 二 種類 が 記さ れ て い 

ます 。 これ ら は 、 古代 の 稲作 を 中心 と し た 共同体 社会 を 背景 に 生まれ た 概念 です 。 



天 つ 罪 ： 共同体 の 根幹 を 破壊 する 神聖 な 罪 



「 天 つ 罪 」 は 、 神話 に おい て 、 高天原 と いう 神々 の 世界 で 素戔嗚 尊 が 犯し た と さ れる 行為 に 由来 し ます 。 その 内容 は 、 共同体 の 生命線 で ある 稲作 を 妨害 する 、 極めて 重大 な 罪 

と さ れ て い まし た 。 



これ ら は 単なる 個人的 な トラブル で は なく 、 コミュニティ 全体 の 生産 活動 や 神聖 な 儀式 を 根本 から 覆し 、 秩序 その もの を 破壊 する 行為 です 。 だ から こそ 、 「 天 つ 」 と いう 言葉 

が 示す 通り 、 天上 の 神聖 な 領域 を 侵す 罪 と し て 、 非常 に 重く 捉え られ て い た の です 。 



国 つ 罪 ： 地上 で 起こる 現実的 な 罪 と 穢れ 



一方 で 、 「 国 つ 罪 」 は 、 われわれ が 住む この 地上 世界 で 発生 する 、 より 現実的 で 身近 な 罪 や 穢れ （ けがれ ） を 指し ます 。 



注目 す べき は 、 ここ に は 現代 で いう 刑法犯 だけ で なく 、 病気 や 天災 、 さらに は 呪い と いっ た 、 本人 の 意図 と は 無関係 に 降りかかる 災厄 まで も が 「 ツミ 」 と し て 数え られ て いる 

点 です 。 これ は 、 古代 日本人 に とっ て の 「 ツミ 」 が 、 個人 の 責任 を 追及 する 道徳的 な 罪 、 と いう より も 、 共同体 の 平穏 や 日常 を 乱す すべて の 異常 事態 を 指し て い た こと を 示し 

て い ます 。 意図的 で あろう と なかろう と 、 平穏 な 状態 が 損なわ れ た こと 、 それ 自体 が 祓い清める べき 「 ツミ 」 だっ た の です 。 



2 . 現代 に 蘇る 「 天 つ 罪 」 ～ 見え ない 秩序 へ の 挑戦 ～ 



で は 、 これ ら の 古代 的 な 罪 の 観念 を 、 現代 社会 に 当てはめ て みる と どう なる でしょう か 。 まず 「 天 つ 罪 」 から 考察 し ます 。 その 本質 が 「 共同体 の 生存 基盤 や 根源 的 な 秩序 を 破 

壊 する 行為 」 で ある と 捉える なら ば 、 現代 に 生きる われわれ に とっ て も 、 決して 無関係 で は あり ませ ん 。 



例えば 、 「 畔 放 」 や 「 溝埋 」 と いっ た 農耕 インフラ の 破壊 は 、 現代 に おけ る 電力 網 や 通信網 、 金融 システム に 対する サイバー 攻撃 や テロ 行為 に 置き換える こと が でき ます 。 ひ 

と たび 社会 インフラ が 麻痺 すれ ば 、 われわれ の 生活 は 一瞬 に し て 崩壊 の 危機 に 瀕し ます 。 これ は まさしく 、 現代 社会 の 生存 基盤 を 揺るがす 「 天 つ 罪 」 と 言える でしょう 。 



また 、 「 頻 蒔 」 と いう 、 他人 の 田 を 攪乱 する 行為 は どう でしょう か 。 これ は 、 インターネット 上 に フェイク ニュース や 悪意 ある 情報 を 拡散 さ せ 、 社会 の 合意 形成 を 妨げ 、 人々 

を 分断 さ せる 行為 と 見なす こと が でき ます 。 あるいは 、 他者 が 築き上げ た 技術 や ブランド を 盗用 する 知的 財産権 の 侵害 も 、 公正 な 競争 と いう 市場 の 秩序 を 乱す 、 現代 の 「 頻 蒔 

」 です 。 



さらに 、 「 生剥 」 や 「 屎戸 」 の よう な 、 神聖 な 領域 へ の 冒涜 行為 は 、 生命 倫理 の 根幹 を 揺るがす よう な 遺伝子 操作 の 濫用 や 、 地球 環境 の 回復 能力 を 無視 し た 大規模 な 環境 破壊 

と し て 捉え直す こと が でき ます 。 これ ら は 、 われわれ 人間 が 踏み込ん で は なら ない 領域 、 すなわち 自然 の 摂理 や 生命 の 尊厳 と いう 「 天 」 の 秩序 を 侵す 行為 に 他 なり ませ ん 。 

で は 、 これ ら の 現代 版 「 天 つ 罪 」 を 、 誰 が どの よう に 判断 する の でしょう か 。 



古代 に おい て は 、 その 判断 は 神 の 意志 を 体現 する 為政者 や 共同体 の 長 に 委ね られ て い まし た 。 それ に 倣う と する なら ば 、 現代 で は 、 まず 第 一 に 国家 や 国際 社会 が 定める 法律 が 

その 役割 を 担い ます 。 サイバー 犯罪 を 取り締まる 法律 や 、 環境 保護 に 関する 国際 条約 が 、 具体的 な 判断 基準 と なり ます 。 



しかし 、 法律 だけ で は 追いつか ない 領域 も 存在 し ます 。 特に 、 生命 倫理 や 最先端 科学 の 分野 で は 、 法整備 が 常 に 後追い に なる の が 現実 です 。 そこ で 重要 に なる の が 、 科学者 コ 

ミュニティ や 業界 団体 が 定める 倫理 規定 、 そして 社会 全体 の 世論 や 良心 と いっ た 、 より 広い 意味 で の 「 公 の 判断 」 です 。 



天 つ 罪 の 本質 が 、 個人 の 領域 を 超え た 「 公 の 秩序 」 の 破壊 で ある 以上 、 その 判断 も また 、 個人 の 感情 や 利害 を 超え た 、 公的 な 視点 から 行わ れる 必要 が ある の です 。 

さらに は 、 その 「 公 」 の 判断 と いう の は 日本人 に 広く 共有 さ れる 常識的 な 判断 と いう こと に も 繋がっ て いく でしょう 。 



3 . われわれ の 内 なる 「 国 つ 罪 」 ～ 病 、 人間 関係 、 そして 災害 ～ 



次 に 、 より われわれ の 日常 に 深く 関わる 「 国 つ 罪 」 に つい て 、 現代的 な 視点 から 考察 を 深め て いき ましょう 。 「 国 つ 罪 」 は 、 個人 の 身体 や 人間 関係 、 そして 自然 の 猛威 と いっ 

た 、 われわれ の 「 生 」 に 直接 降りかかる 苦悩 や 過ち を 映し出し て い ます 。 



言葉 の 刃 と 心 の 傷 



「 生 膚 断 」 は 、 人 の 身体 を 傷つける 行為 です 。 これ は 現代 の 刑法 に おけ る 傷害罪 に 直結 し ます が 、 その 射程 は 物理的 な 暴力 に 限り ませ ん 。 インターネット や SNS が 普及 し た 

現代 に おい て 、 匿名 で 投げかけ られる 誹謗 中傷 や 、 職場 や 学校 で 繰り返さ れる ハラスメント と いっ た 「 言葉 の 暴力 」 は 、 人 の 心 を 深く 傷つけ 、 時 に は 物理的 な 傷 以上 に 深刻 な 

ダメージ を 与え ます 。 これ も また 、 現代 に おけ る 「 生 膚 断 」 で あり 、 祓わ れる べき 重大 な 「 ツミ 」 です 。 



信頼 関係 の 崩壊 



「 己が 母 犯せる 罪 」 など に 代表 さ れる 近親 相姦 の 罪 は 、 家族 と いう 最も 基礎的 な 人間 関係 の 秩序 を 破壊 する 行為 です 。 これ を 現代的 に 拡張 し て 考える なら ば 、 家族 や 友人 、 恋 

人 と いっ た 親しい 間柄 に おけ る 裏切り や 、 信頼 関係 を 根底 から 覆す よう な 不誠実 な 行為 全般 が 、 これ に あたる と 言える でしょう 。 法的 に 罪 に 問わ れる か どう か と は 別 に 、 人 と 

人 と の 絆 を 断ち切り 、 コミュニティ の 健全性 を 損なう 行い は 、 祓い清める べき 「 ツミ 」 な の です 。 



病 や 災害 と の 向き合い 方 



大 祓詞 が 、 病気 （ 白人 ・ 胡 久美 ） や 天災 （ 高津 神 の 災 ） まで も 「 ツミ 」 に 含め て いる 点 は 、 現代 の われわれ に とっ て 非常 に 示唆 的 です 。 もちろん 、 現代 に おい て 病気 や 災害 に 

遭っ た 人 を 「 罪人 」 と 見なす こと は あり ませ ん 。 医療 や 科学 の 発展 に より 、 これ ら は 克服 ・ 予防 す べき 対象 へ と 変わり まし た 。 



しかし 、 この 祝詞 は 、 われわれ が 病 や 災害 と いう 「 不可抗力 」 と いかに 向き合う べき か を 問いかけ て い ます 。 



重要 な の は 、 病気 や 障害 を 持つ 人々 に 対する 差別 や 偏見 が 、 まさに 現代 社会 に おけ る 「 ツミ 」 と し て 存在 し て いる と いう 事実 です 。 



また 、 自然 災害 に 対し て 、 日頃 の 備え を 怠る 「 人災 」 と し て の 側面 や 、 被災地 で の デマ の 拡散 、 支援 を 妨害 する よう な 行為 は 、 われわれ 自身 が 生み出す 二 次 的 な 「 ツミ 」 と 言 

え ます 。 



で は 、 これ ら の 個人的 ・ 内面的 な 領域 に 深く 関わる 「 国 つ 罪 」 は 、 誰 が 判断 する の でしょう か 。 

ここ で も 、 傷害罪 を 裁く 法律 や 、 医療 過誤 を 判断 する 医療 倫理 と いっ た 公的 な 基準 が まず 存在 し ます 。 しかし 、 「 国 つ 罪 」 の 多く は 、 法律 だけ で は 裁き きれ ない 心 の 領域 に ま 

で 及ん で い ます 。 



言葉 の 暴力 や 人間 関係 の トラブル は 、 最終的 に は 当事者 間 の 対話 や 、 コミュニティ に よる 調停 に よっ て 解決 が 図ら れる べき もの です 。 そして 、 何 より も 重要 な 判断 者 は 、 われ 

われ 一人ひとり の 内 なる 良心 な の です 。 



自ら の 言動 が 他者 を 傷つけ て い ない か 。 人 と し て の 道 を 踏み外し て い ない か 。 自然 の 恵み や 脅威 に 対し 、 謙虚 な 心 で 向き合え て いる か 。 常 に 自ら を 省みる 内省 的 な 姿勢 こそ が 

、 「 国 つ 罪 」 を 判断 し 、 それ を 乗り越え て いく ため の 鍵 と なる の です 。 



4 . 本居 宣長 が 解き明かし た 「 ツミ 」 の 真相 



ここ まで 、 「 天 つ 罪 」 「 国 つ 罪 」 を 現代 に 置き換え て 考察 し て き まし た が 、 そもそも 古代 日本人 は 「 ツミ 」 と いう もの を 、 どの よう に 捉え て い た の でしょう か 。 この 根源 的 な 

問い に 、 鮮やか な 光 を 当て た の が 、 江戸 時代 の 国学者 ・ 本居 宣長 です 。 

宣長 は 、 その 主著 の 一 つ で ある 『 大 祓詞 後 釈 （ お お はらえ の ことば のち の しゃく ） 』 の 中 で 、 「 ツミ 」 の 語源 に つい て 画期的 な 解釈 を 提示 し まし た 。 



「 ツミ 」 は 「 包み 」 で ある 



宣長 は 、 多く の 学者 が 「 ツミ 」 を 道徳的 な 罪悪 と 考え て い た の に 対し 、 その 語源 は 「 包（つつむ） 」 と いう 言葉 に ある と 論じ まし た 。 つまり 、 「 ツミ 」 と は 、 何 か が 「 包み 隠 

さ れ た 状態 」 を 指す と いう の です 。 



で は 、 何 が 包み隠さ れ て いる の か 。 



それ は 、 「 ケガレ （ 穢れ ） 」 です 。 



「 ケガレ 」 と は 、 死 や 病 、 不浄 な もの に 触れる こと で 生じる 、 良く ない 気 、 あるいは 生命力 の 減衰 し た 状態 を 指し ます 。 宣長 に よれ ば 、 「 ツミ 」 と は 、 人 の 心身 や 生活 の 場 が 

、 この 「 ケガレ 」 に よっ て 覆わ れ 、 包み隠さ れ て しまい 、 本来 ある べき 清浄 な 状態 が 失わ れ て いる こと を 意味 し ます 。 



この 解釈 は 、 西洋 的 な 罪 の 概念 、 特に キリスト教 に おけ る 「 Sin （ シン ） 」 と は 全く 異なる 思想 です 。 西洋 の 「 Sin 」 は 、 神 が 定め た 絶対的 な 戒律 に 背く 行為 で あり 、 そ 

こ に は 人間 の 原罪 や 、 本性 の 悪 と いっ た 観念 が 根底 に あり ます 。 つまり 、 罪 は 罰せ られる べき 「 悪 」 な の です 。 



しかし 、 宣長 の 解釈 する 「 ツミ 」 は 、 本性 が 悪 だ から 犯す もの で は あり ませ ん 。 われわれ は 本来 、 清らか な 存在 で ある に も かかわら ず 、 生活 する 中 で 意図 せ ず 「 ケガレ 」 に 触 

れ て しまう こと が ある 。 その 結果 と し て 、 知ら ず 知ら ず の うち に 「 ツミ 」 の 状態 に 陥っ て しまう 、 と 考え ます 。 それ は 、 ホコリ が 積もる よう に 、 自然 に 付着 し て しまう もの に 

近い 感覚 です 。 



祓い と は 「 元 の 状態 に 戻す 」 こと 



「 ツミ 」 が 「 包ま れ た 状態 」 で ある なら ば 、 「 ハラヒ （ 祓い ） 」 と は 何 を 意味 する の でしょう か 。 それ は 、 その 包み を 取り払い 、 中 の 本来 の 姿 を 現れ させる 行為 です 。 積もっ 

た ホコリ を 掃き清める よう に 、 ケガレ を 祓い 、 清浄 な 元 の 状態 に 復帰 さ せる こと 。 それ が 「 祓い 」 の 本質 です 。 

ここ に 、 罰 に よっ て 罪 を 償わ せる と いう 思想 と は 対照的 な 、 「 回復 」 と 「 再生 」 の 思想 が 見 て 取れ ます 。
