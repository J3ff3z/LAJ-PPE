なずな ず 日記 



昔 から ＆ 日頃 から 気 に なっ てる 事 を 徒然 なる まま に だらだら と 書き綴っ た エッセイ 風 blog です 。 







天 つ 罪 



神社 で 神主 が 唱える 「 大祓 ( お お はらえ ） 」 の 「 祝詞(のりと） 」 の 中 に 、 「 天 つ 罪 ( あ ま つつみ ） 」 「 国 つ 罪 ( くに つつみ ） 」 と いう 言葉 が 出 て き ます 。 

「 天 つ 罪 」 は 、 『 古事記 』 や 『 日本 書紀 』 に 記さ れる 、 スサノオノミコト が 高天原 で 犯し た 行為 に 由来 する と さ れ て い ます 。 

犯し たら 太陽 が 出 て 来 なく なる よう な 重い 罪 です 。 

「 畔 放(あはなち） 」 「 溝埋(みぞうめ） 」 「 樋 放(ひはなち） 」 「 頻 播(しきまき） 」 「 串刺(くしさし） 」 など の ５ つ は 、 稲作 の 妨害 を 罪 と する もの です 。 

田 に 引い た 水 を 故意 に 外 に 流し たり 、 他人 の 田畑 に 上 から 種 を 蒔い たり し て は いけ ない 、 と いう 戒め です 。 

稲作 を 大切 に する 日本人 らしい 罪 の 意識 だ と 思い ます 。 

もう 一 つ 、 「 糞(くそへ） 」 と いう 、 いかに も 不浄 な 罪 が あり ます (^_^;) 。 

神聖 な 場所 を 汚物 で 汚し て は いけ ない と いう 戒め です 。 

さて 、 いきなり 私 が 「 天 つ 罪 」 に つい て 書い た の は 、 残る ２ つ の 罪 に つい て お 伝え し たかっ た から です 。 

残る ２ つ は 、 「 生剥(いきはぎ） 」 と 「 逆剥(さかはぎ） 」 です 。 

「 生剥 」 と は 、 生き た まま 動物 の 皮 を 剥ぐ 罪 、 あるいは 動物 の 皮 を 剥い で 殺す 罪 です 。 

「 逆剥 」 と は 、 既に 死ん で いる 動物 の 皮 を 剥ぐ 時 、 正しい 方向 ( 頭 → 尻 ） で は なく 、 逆 ( 尻 → 頭 ） に 剥ぐ 罪 です 。 

犯す と 太陽 が 輝か なく なっ て しまう ほど の 重い 罪 と し て 、 動物 に 苦痛 を 与える こと を 禁じ て いる 古代 日本人 の 心 に 、 私 は 感動 し て しまい ます 。 

冬 に なる と 、 ファー を オシャレ な 小道具 に 使っ た ファッション が 増え て 、 見 て い て 辛く なり ます (>_<) 。 

動物 に 恐怖 と 苦痛 の 極み を 与え ながら 、 血しぶき と 悲鳴 の 中 で 剥ぎ取ら れ た 毛皮 は 、 この世 で １番 可哀相 な 動物 の 屍 に しか 見え ませ ん 。 

欧米 で は 、 既に リアル ファー を 止める ムーブメント が 起き て い ます 。 

しかし 、 日本 で は 、 まだ 冬 の ファッション アイテム と し て もてはやさ れ て い ます 。 

「 国 つ 罪 」 は 、 世の中 に 汚れ を 生じ 、 天変 地異 を   引き起こす 原因 に なる 罪 です 。 

ある 種 の 病気 や 雷 など 、 現代人 に は 全く 罪 と は 思え ない もの も あり ます が 、 傷害罪 ・ 傷害 致死 罪 や 近親 相姦 の 罪 など も 含ま れ ます 。 

その 中 に も 「 畜 たおし ( けもの たおし ） 」 と いう 罪 が あり ます 。 

家畜 を 殺傷 する 罪 です 。 

古代 日本人 が 動物 を どれ だけ 大切 に し て い た か が わかり ます 。 

キリスト教 に は 「 七 つ の 大罪 」 と いう 罪 の 考え が あり ます 。 

「 傲慢 」 「 嫉妬 」 「 憤怒 」 「 怠惰 」 「 強欲 」 「 暴食 」 「 色欲 」 です 。 

犯罪 と いう より 、 精神 の 持ち 様 を 示し て いる よう です 。 

仏教 に も 「 八戒 斎 」 と いう 戒め が あり ます 。 

「 不 殺生 戒 」 「 不 偸盗 戒 」 「 不 婬 戒 」 「 不 妄語 戒 」 「 不 飲酒 戒 」 「 不 香油 塗 身 戒 」 「 不 歌舞 観 聴 戒 」 「 不 高 広大 床 戒 」 。 

お 酒 を 飲ま ない 、 広く ゆったり し た ベッド で 寝 ない …   など 、 修行 を する 出家 者 の タブー を 決め て い ます 。 

キリスト教 も 仏教 も 、 それぞれ の 宗教 に おい て 信徒 に 求める こと を 決め て い ます が 、 同じ 宗教 で も 日本 の 神道 は 人間 と 自然 、 人間 と 動物 と の 正しい 付き合い 方 に 言及 し て いま 

す 。 

私 は 、 神道 の そう いう 考え が 好き です 。 

それ が 大和魂 だ と 思い ます 。 

いにしえ の 日本人 の 心 に 立ち返り 、 動物 虐待 も 残酷 な 作り方 を し た 毛皮 の 購入 ( 輸入 ） も 無い 国 に し たい です 。
