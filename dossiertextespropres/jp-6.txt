「 七 つ の 大罪 」 って ご存知 です か ？ 



7 つ すべて を 言える 方 や 、 具体的 な 内容 に つい て 説明 できる 方 は 、 案外 少ない か も しれ ませ ん 。 



そこ で 今回 は 、 七 つ の 大罪 と は どんな もの な の か 、 罪 の 重い 順 に 解説 し ます 。 



また 、 それぞれ の 罪 を 象徴 する 悪魔 や 動物 、 新 ・ 七 つ の 大罪 に つい て も ご 紹介 し ます 。 

目次 



七 つ の 大罪 と は ？ 種類 や 起源 を 解説 



七 つ の 大罪 と は 、 「 傲慢 」 「 嫉妬 」 「 憤怒 」 「 怠惰 」 「 強欲 」 「 暴食 」 「 色欲 」 と いう 7 つ の こと 。 



キリスト教 の 西方 教会 、 主に カトリック 教会 に おけ る 用語 で 、 意味 は 英語 や ラテン語 で 「 七 つ の 死 に 至る 罪 」 です 。 



ただ 、 罪 その もの と いう より も 、 人間 を 罪 に 導く 可能性 が ある 欲望 や 感情 の こと を 指す ため 、 日本 の カトリック 教会 で は 「 七 つ の 罪 源（ざいげん） 」 と 訳し ます 。 



簡単 に 言う と 、 「 これ ら 7 つ の 欲望 や 感情 に 身 を 任せる と 、 自分 や 人 を 死 に 至ら しめる ほど の 罪 を 犯す 危険性 が ある 」 と いう こと な の です ！ 



起源 は 、 4 世紀 エジプト の 修道士 、 エヴァグリオス ・ ポンティコス の 著者 『 修行 論 』 に 、 八 つ の 「 人間 一般 の 想念 」 と し て 現れ た こと 。 



八 つ の 想念 と は 、 「 貪食 （ 暴食 ） 」 「 淫蕩 （ 色欲 ） 」 「 金銭 欲 （ 強欲 ） 」 「 憂鬱 」 「 怒り （ 憤怒 ） 」 「 怠惰 ） 」 「 虚飾 （ 虚栄 ・ 自惚れ ） 」 「 傲慢 」 です 。 



これ ら が 6 世紀 後半 、 ローマ 教皇 グレゴリウス 1 世 に よっ て 、 「 虚飾 」 が 「 傲慢 」 に 含ま れ 、 「 怠惰 」 と 「 憂鬱 」 が 一 つ に 、 そして 「 嫉妬 」 が 追加 さ れ 、 現在 の 「 七 つ の 大罪 

」 に なり まし た 。 



七 つ の 大罪 ｜ 罪 の 重さ に 順番 は ある ？ 



「 七 つ の 大罪 （ Seven   Deadly 

Sins ） 」 に は 、 伝統的 に 明確 な 「 罪 の 重さ の 順番 」 は 存在 し ませ ん が 、 一部 の 神学者 や 宗教的 伝統 で は 、 罪 の 重さ に 差 が ある と 考え られ て き まし た 。 



特に キリスト教 の カトリック 神学 で は 、 罪 に は 「 大罪 （ mortal   sin ） 」 と 「 小 罪 （ venial 

sin ） 」 が ある と さ れ 、 その 中 で も 七 つ の 大罪 は 人 を 堕落 さ せる 根本的 な 「 根源 的 な 罪 」 と みなさ れ ます 。 



以下 は 七 つ の 大罪 の 歴史的 ・ 神学 的 な 一部 解釈 に 基づい た 「 重い 順 」 と さ れる こと の ある 順番 です 。

七 つ の 大罪 に は 、 それ ら を 司る 悪魔 や 象徴 する 幻獣 ・ 動物 が ある ん です ！

七 つ の 大罪 の 具体的 な 意味 や 内容 を 解説 ！ 



七 つ の 大罪 を 司る 悪魔 や 、 象徴 する 幻獣 ・ 動物 たち を ご 紹介 し まし た が 、 悪魔 や 幻獣 の 名前 は なかなか 聞か ない こと でしょう 。 



彼 ら は いったい 何者 な の か 、 その 正体 が 気 に なる 方 も いる か と 思い ます 。 



そこ で ここ で は 、 それ ら の 正体 を 含め 、 7 つ の 大罪 が どの よう に 人 の 行動 や 社会 に 影響 を 与える の か など 、 具体的 な 内容 や 意味 を 罪 の 重い 順 に 解説 し て いき ます 。 



① 傲慢 



最も 罪深い と さ れる 「 傲慢 」 は 、 自己 中心 的 な 態度 と 他人 へ の 無関心 を 言い 、 自分 を 他人 より も 高く 評価 する こと から 、 他者 と の 孤立 や 対立 を 引き起こし ます 。 



傲慢 を 司る 悪魔 は ” ルシファー “ 。 



ルシファー は 、 悪魔 の 王 や 地獄 の 王 と し て 、 数 いる 悪魔 の 中 で も 最強 の 力 を 持つ 存在 です 。 



「 光 を もたらす 者 」 と いう 意味 の ルシファー は 、 元々 は 神 に 仕える 天使 の 長 で 、 天使 の 中 で も 最高 の 能力 と 地位 の ある 大天使 でし た 。 



しかし 、 神 に 反逆 し 「 闇落ち し た 堕天 使 、 サタン 」 に なっ て しまっ た の です 。 



元々 は 12 枚 の 翼 を 持つ 神 の 右腕 と 称さ れる 、 天使 の 中 で も 最も 美しい トップ クラス の 大天使 だっ た よう です ね 。 



幻獣 は ” グリフォン “ 。 



グリフォン と は 神話 や 伝説 に 登場 する 生物 で 、 ワシ の 頭 と 翼 、 ライオン の 身体 を 持つ 姿 を し 、 力強 さ や 守護 、 神秘 性 の 象徴 と し て 知ら れ て い ます 。 



ワシ は 空 、 ライオン は 地上 に おい て 「 王 」 する こと から 、 グリフォン を 「 天 と 地 を 支配 する 力 」 の 象徴 と 認識 し 、 強大 な 存在 と し て の イメージ を 確立 さ れ て い ます 。 



② 嫉妬 



傲慢 に 次ぐ 罪 の 重さ で ある 「 嫉妬 」 は 、 他人 の 成功 や 幸福 を 羨ん だり 妬ん だり する 感情 で 、 心 や 社会 の 和 を 乱し 人間 関係 の 軋轢 の 原因 と なり 得 ます 。 



嫉妬 を 司る 悪魔 は ” レヴィ アタン “ 。 



レヴィ アタン は 、 旧約 聖書 に 登場 する 最強 の 海 の 悪魔 （ 怪物 ） と し て 有名 です 。 



旧約 聖書 で は 、 神 が 天地 創造 の 5 日 目 に 作り出し た 存在 と し て 記さ れ て おり 、 同じく 神 に 作ら れ た ベヒモス と 二 頭 一 対 を 成す と さ れ て い ます 。 



また レヴィ アタン は 、 その 硬い 鱗 と 巨大 さ から 、 いかなる 武器 も 通用 し ない 最強 の 生物 と 言わ れ 、 彼 が 海 を 泳ぐ と 波 が 逆巻き 、 鼻 から は 煙 を 、 口 から は 炎 を 吐き 、 凶暴 で 冷酷 

無比 な の だ そう 。 



繁殖 を 防ぐ ため 雄 しか 存在 せ ず 、 大 ウソつき 者 で 不死身 です 。 



幻獣 は ” マーメイド “ 。 



童話 を はじめ 、 さまざま な 作品 や キャラクター の モチーフ と さ れる マーメイド は 、 今 や その 名 を 知ら ない 人 は い ない でしょう 。 



マーメイド は 具体的 に 、 上半身 が 人間 で 下半身 が 魚 の 形 を し た 想像 上 の 女 の 人魚 を 指し 、 一説 に は ジュゴン や マナティ を モデル と し 、 マーメイド を 食す と 若返る と いう 逸話 が 

あり ます ね 。 



一般的 に は 人間 と 同じ サイズ です が 、 時 に は 600 m を 超える なんて いう 民話 も …   。 



③ 憤怒 



「 憤怒 」 は 理性 を 失い 感情 の コントロール が でき ない こと から 、 しばしば 暴力 や 衝突 を 引き起こし ます 。 



憤怒 を 司る 悪魔 は ” サタン “ 。 



サタン は ヘブライ 語 で 、 「 敵対者 」 「 訴える 者 」 「 妨げる 者 」 など を 意味 し ます 。 



かつて は ” ルシファー ” と し て 、 キリスト教 ・ ユダヤ教 で 神 に 仕える 御 使い でし た が 、 神 に 反逆 し て から は 「 神 の 敵対者 」 と し て 、 悪魔 に 変化 し て しまい まし た 。 



また 、 イスラム教 で も 人間 の 敵 と さ れ て いる の です 。 



幻獣 は ” ユニコーン “ 



ユニコーン は ヨーロッパ の 伝説 上 の 生き物 で 、 馬 の 体 と 額 に は ねじれ た 1 本 の 角 を 持つ 「 一角獣 」 です 。 



角 に は 解毒 する 力 が ある と する 説 も あり ます が 、 一般的 に は 真っ白 な 馬 に パステル カラー が 立て 髪 や しっぽ に 使わ れる かわいらしい キャラクター と ファンタジー な 世界観 と し 

て 、 女の子 に 大人気 です ね 。 



④ 怠惰 



「 怠惰 」 と は 行動 を 起こす こと へ の 無関心 や 意欲 の 欠如 を 言い 、 個人 の 成長 や 社会 貢献 の 妨げ と なり ます 。 



怠惰 を 司る 悪魔 は ” ベルフェゴール “ 。 



ベルフェゴール と は 、 「 ぺ オル 山 の 主神 」 を 意味 し 、 その 姿 は 牛 の 尻尾 に ねじれ た 2 本 の 角 、 顎 に は ひげ を 生やし た 醜悪 な もの と さ れ て い ます 。 



人間 に 便利 な 発明 品 を 与え て 、 怠惰 の 罪 に 陥れる 悪魔 な の だ そう 。 



また 、 人間 の 結婚 生活 を 覗き見る 悪魔 ともさ れ て おり 、 女性 の 心 に 性的 で 不道徳 な 心 を 芽生え させる 力 を 持つ ため か 、 女性 に 対し て 不信感 を 持っ て い た と さ れ て い ます 。 



幻獣 は ” フェニックス “ 



フェニックス は 、 死ん で も 蘇る こと で 永遠 の 時 を 生きる と さ れる 伝説 の 鳥 です 。 



寿命 を 迎える と 自ら 炎 に 飛び込ん で その 命 を 終え ます が 、 再び 若い 姿 に なっ て 蘇る こと から 「 不死鳥 」 と も 言わ れ ます 。 



⑤ 強欲 



「 強欲 」 は 過度 の 物質的 欲求 や 財産 の 貯蓄 を 指し 、 社会的 不平等 や 環境 問題 に 繋がり ます 。 



強欲 を 司る 悪魔 は ” マモン “ 。 



一般的 に 「 富 の 悪霊 」 と し て 擬人化 し た もの と 言わ れ て いる マモン は 、 「 地獄 の 七 大 君主 」 の 一人 ともさ れ て い ます 。 



地獄 の 七 大 君主 と は 、 地獄 の 貴族 の 代表格 と さ れ 、 地 に 落ち た 神 の 影 、 強壮 な 天使 の 一群 の こと を 表し て い ます 。 



マモン は 時 に 、 袋 を 抱え て 箱 に 腰 を 下ろす 男 の 姿 や 、 金貨 を ばらまく 悪魔 の 姿 で 現さ れる こと が ある よう です 。 



ジョン ・ ミルトン の 『 失 楽園 』 に よる と 、 マモン は 堕天 使 の 中 で 最も 卑しい 根性 の 持ち主 で 、 神 より も 下界 の 金 着い 財宝 を 賛美 し て い た と さ れ て い ます 。 



幻獣 は ” コブリン “ 



ヨーロッパ の 民間 伝承 に 登場 する 伝説 の 生き物 ・ コブリン は 、 人 の 膝 ほど の 背丈 と 灰色 の 髪 や あごひげ を 持つ 妖精 です 。 



普段 は 洞穴 や 木立 に 住み 、 小さな 子供 が 大好き ！ 



良い 子 に は プレゼント を くれ たり 面倒 を 見 て くれ ます が 、 同時 に 人間 の 家 を 引っ搔き 回す など の 悪戯 を する こと も 。 



また 時 に 、 小さな 子供 を 食べる 邪悪 な 者 と さ れ て い ます 。 



⑥ 暴食 



「 暴食 」 は その 名 の 通り 過度 な 食欲 を 指し 、 健康 問題 や 資源 の 無駄遣い の 要因 と なり ます 。 



暴食 を 司る 悪魔 は ” ベルゼブブ “ 。 



ベルゼブブ は 、 ヘブライ 語 で 「 ハエ の 王 」 を 意味 し ます が 、 本来 は ” バアル ・ ゼブル ” と いい 、 「 気 高き 主 」 と いう 意味 です 。 



地獄 に おい て サタン に 次ぐ 権力 と 邪悪 さ だ と さ れ 、 その 実力 は サタン を も しのぐ ほど の 魔王 だ と も 言わ れ て い ます 。 



また ベルゼブブ は 、 作物 を 荒らす ハエ の 害 から 人間 を 救う 力 も 持っ て おり 、 神託 を もたらす 悪魔 だ と さ れ て いる の です 。 



幻獣 は ” ケルベロス “ 



ケルベロス は ギリシャ 神話 に 登場 する 犬 の 怪物 で 、 ハーデース が 支配 する 冥界 の 番犬 。 



一般的 に は 3 つ の 頭 を 持ち 、 普段 は 1 頭 ずつ 交代 で 眠り 残り 2 頭 が 常 に 見張り を し て い ます 。 



ハーデース に 非常 に 忠実 で 、 死者 の 魂 が 冥界 やっ て くる と そのまま 通し ます が 。 冥界 から 逃げ出そう と する 者 は すぐさま 捉え て 食べ て しまう の だ そう 。 



⑦ 色欲 



「 色欲 」 は 不適切 な 性的 欲望 の こと で 、 人間 関係 の 破綻 や 倫理的 問題 を 引き起こし ます 。 



色欲 を 司る 悪魔 は ” アスモデウス “ 。 



元 は 智 天使 で ある 堕天 使 で 、 その 容姿 は 「 人 」 「 牛 」 「 羊 」 の 3 つ の 頭 と ガチョウ の 足 、 毒蛇 の 尻尾 を 持ち 、 手 に は 軍旗 と 槍 を 持っ て い ます 。 



そして 、 地獄 の 龍 に 跨り 、 口 から 火 を 噴い て いる 恐ろしい 悪魔 と さ れ て い ます 。 



アマイモン 配下 の 東宝 の 悪魔 の 首座 で あり 、 72 の 軍団 を 率いる 序列 32 番 の 偉大 な 王 な の です 。 



幻獣 は   “ サキュバス “ 



サキュバス は 、 男性 を 魅了 する 悪魔 で 、 理想 の 女性 の 姿 を し た 夢魔 （ 下級 の 悪魔 ） の こと 。 



反対 に 女性 を 魅了 する 悪魔 を ” インキュバス ” と 言い 、 理想 の 外見 を し た 男性 の 夢魔 で 、 悪魔 の 子 を 妊娠 さ せ ます 。 



新 ・ 七 つ の 大罪 を 紹介 



実 は 、 2008 年 に バチカン の ローマ 教皇 庁 より 、 「 新 ・ 七 つ の 大罪 」 が 発表 さ れ まし た 。 



その 理由 は 、 「 従来 の 七 つ の 大罪 は 、 個人 主義 的 な 側面 が あり 、 それ と は 異なる 種類 の 罪 も ある と いう こと を 信者 に 伝え 、 理解 さ せる ため 」 と し て い ます 。 



新 ・ 七 つ の 大罪 は 、 以下 の 通り です 。 



1 .   遺伝子 を 改造 する こと 

2 .   人体 実験 を 行う こと 

3 .   環境 を 汚染 する こと 

4 .   社会的 な 不公平 を 行う こと 

5 .   他人 を 貧困 に する こと 

6 .   悪辣 に 金 を 得る こと 

7 .   薬物 を 濫用 する こと 



現代 に おい て 非常 に 分かり やすく 、 そして 誰 も が 納得 の いく もの と なっ て い ます ね 。 



また 、 インド 独立 の 父 と し て 知ら れる 「 マハトマ ・ ガンディー 」 は 、 20 世紀 資本 主義 に おけ る 「 七 つ の 社会的 大罪 」 を 提唱 おり 、 ガンディー の 慰霊碑 に も 刻ま れ て い ます 。 



以下 が 、 彼 の 提唱 し た 七 つ の 社会的 大罪 です 。 



1 .   原則 なき 政治 

2 .   道徳 なき 商業 

3 .   労働 なき 富 

4 .   人格 なき 教育 

5 .   人間性 なき 科学 

6 .   良心 なき 快楽 

7 .   献身 なき 信仰 



こちら も 従来 の 七 つ の 大罪 と は 異なる 視点 で 、 現代 に 即し た 内容 と なっ て いる と 言える でしょう 。 



まとめ 



「 七 つ の 大罪 」 の 意味 や 罪深 さ に つい て ご 紹介 し まし た 。 



きっと 初めて 知る 悪魔 や 幻獣 も い た こと でしょう 。

「 大罪 」 と 言わ れ て いる ほど です から 、 十分 に 気 を 付け て 生活 し て いき たい です ね 。
